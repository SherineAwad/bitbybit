Installation Guide¶

rvtests¶

Install rvtests from https://github.com/zhanxw/rvtests

git clone https://github.com/zhanxw/rvtests

lncScore¶

Install lncScore https://www.nature.com/articles/srep34838

wget https://github.com/WGLab/lncScore/archive/v1.0.2.tar.gz
tar -xzvf v1.0.2.tar.gz

Trimgalore¶

Install Trimgalore https://www.bioinformatics.babraham.ac.uk/projects/trim_galore/

conda install -c bioconda trim-galore

Cutadapt¶

Install Cutadapt http://cutadapt.readthedocs.io/en/stable/installation.html

conda install -c bioconda cutadapt

FastQC¶

Install FastQC https://www.bioinformatics.babraham.ac.uk/projects/fastqc/

conda install -c bioconda fastqc

BBMap¶

Lets downlowd BBMap source

wget https://sourceforge.net/projects/bbmap/files/BBMap_38.07.tar.gz

Then lets unzip the files

tar -xzvf BBMap_38.07.tar.gz

BBMap scripts are ready to use in bbmap directory

cd bbmap

vcftools¶

We will use conda

conda install vcftools

Plink¶

conda install -c biobuilds plink

Samtools¶

conda install -c bioconda samtools

Salmon¶

conda install salmon

Tophat¶

conda install -c bioconda tophat
